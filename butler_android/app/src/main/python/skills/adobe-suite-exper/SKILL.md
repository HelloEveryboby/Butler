## 一、Photoshop —— 图像合成与像素处理

### 1.1 非破坏性编辑黄金法则
- **调整层** 代替直接调色
- **智能对象** 允许反复缩放与滤镜
- **蒙版** 代替橡皮擦，所有显隐均可逆
- **盖印层** `Ctrl+Shift+Alt+E` 用于合并快照，保留原始图层

### 1.2 高精度抠图决策
| 背景特征 | 推荐工具 | 适用版本 |
|----------|---------|---------|
| 主体与背景清晰分离 | 对象选择工具 + 选择主体 | CC 2020+ |
| 半透明/毛发 | 选择并遮住 > 调整边缘画笔 | CC 2015.5+ |
| 纯色背景 | 魔棒 + 图层蒙版 + 边缘平滑 | 全版本 |
| 复杂边缘 | 钢笔工具 + 路径转选区 | 全版本 |
| 极致通道 | 通道面板找高对比通道 + 色阶 | 全版本 |

### 1.3 人像精修五步法
1. **液化**（滤镜 > 液化）：微调轮廓，注意五官比例
2. **去瑕疵**：新建空白层，污点修复画笔，样本设为“所有图层”
3. **皮肤柔化**：盖印层 → 表面模糊（阈值15~20）→ 黑色蒙版，白色画笔擦皮肤
4. **光影重塑**：中性灰图层（柔光模式），黑白画笔加深减淡
5. **锐化输出**：USM 锐化（数量80%，半径1.5）只作用于眼睛睫毛区域

## 二、Illustrator —— 矢量图形与设计

### 2.1 工作区与核心工具
- **基本图形工具**：矩形(M)、椭圆(L)、钢笔(P)、曲率工具(Shift+~)
- **路径查找器**（窗口 > 路径查找器）：合并、减去顶层、交集等
- **形状生成器**（Shift+M）：直观合并和分割形状
- **宽度工具**（Shift+W）：自由调整描边粗细
