# Anthropic 技能创作助手 (Anthropic Skill Creator)

这是从 Anthropic 官方引入的专业级技能开发工具集，专门用于创建、测试和优化 Butler 技能。

## 功能特性

- **结构化开发**：遵循 `SKILL.md` + `scripts/` + `references/` 的标准结构。
- **自动化测试**：支持批量运行测试用例，并与基准线（Baseline）进行对比。
- **可视化评估**：内置 `eval-viewer` 工具，支持在浏览器中定性评估输出质量。
- **性能分析**：自动统计 Token 消耗、耗时等关键性能指标。
- **描述优化**：通过自动化的迭代循环，优化技能描述以提高触发准确率。

## 目录结构

- `SKILL.md`: 核心操作指南。
- `scripts/`: 功能脚本（评估、聚合、描述优化等）。
- `agents/`: 用于辅助开发的子代理（Subagent）指令。
- `eval-viewer/`: 基于 Web 的评估结果查看器。
- `references/`: 相关的技术参考文档和 Schema 定义。

## 使用方法

1. **阅读指南**：AI 会首先阅读 `SKILL.md` 以了解整个开发工作流。
2. **定义意图**：确定技能的目标、触发场景和预期输出。
3. **编写草案**：创建新的技能文件夹并编写其 `SKILL.md`。
4. **运行测试**：使用 `scripts/run_eval.py` 或 `run_loop.py` 进行自动化测试。
5. **人工复核**：通过 `eval-viewer/generate_review.py` 生成评估页面。
6. **循环迭代**：根据反馈不断优化，直到达到生产级质量。

## 开发建议

- 优先通过 `__init__.py` 引入逻辑，但保持 `SKILL.md` 作为最高层级的操作指令。
- 所有的“为什么”都应该在指令中解释清楚，以便 AI 更好地理解任务意图。
- 善用 `eval-viewer` 来发现细微的质量问题。
