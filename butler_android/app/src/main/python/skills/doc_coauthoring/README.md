# 文档协同创作工作流 (Doc Co-Authoring Workflow)

这是一个旨在通过结构化过程提升文档质量的技能插件。它将文档创作分为背景收集、内容精炼和读者测试三个阶段，确保最终产出的文档既专业又易于理解。

## 主要功能

- **深度背景收集**：通过系统性提问，挖掘项目背景、受众需求及核心目标。
- **章节化精炼**：采用头脑风暴和迭代反馈，逐章构建高质量内容。
- **模拟读者测试**：通过预测问题和盲点检查，验证文档的易读性和逻辑完备性。

## 如何使用

当您需要进行以下创作时，可以调用此技能：
- 编写技术规范、PRD 或设计文档。
- 起草项目提案、决策文档或 RFC。
- 进行任何需要高度逻辑性和清晰表达的深度写作任务。

# Doc Co-Authoring Workflow Skill

This skill plugin is designed to enhance document quality through a structured process. It divides document creation into three stages: Context Gathering, Content Refinement, and Reader Testing, ensuring the final output is both professional and easy to understand.

## Key Features

- **In-depth Context Gathering**: Uncover project background, audience needs, and core goals through systemic questioning.
- **Sectional Refinement**: Build high-quality content chapter by chapter using brainstorming and iterative feedback.
- **Simulated Reader Testing**: Validate readability and logical completeness through question prediction and blind spot checks.

## How to Use

Invoke this skill when you need to:
- Write technical specs, PRDs, or design docs.
- Draft project proposals, decision docs, or RFCs.
- Engage in any deep writing task that requires high logic and clear expression.
