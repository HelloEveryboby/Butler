# 极致前端设计 (Frontend Aesthetics)

此技能旨在引导创建具有独特辨识度、生产级别的优质前端界面，坚决杜绝平庸的“AI 工业废料”式审美。通过对审美细节和创意选择的极致追求，交付真正可运行的高质量代码。

当用户提出前端需求（组件、页面、应用或界面）时，应结合其用途、受众及技术约束进行深度创作。

## 核心设计思维 (Design Thinking)

在开始编码前，必须深入理解上下文，并承诺一个**大胆的审美方向**：

-   **目标 (Purpose)**：这个界面解决什么问题？谁在使用它？
-   **调性 (Tone)**：选择一个极致的方向：极端极简、极繁主义、复古未来感、有机自然、奢华精致、童趣可爱、杂志排版、粗犷主义、装饰艺术/几何、柔和粉调、工业实用主义等。从这些风格中汲取灵感，但要设计出真正符合该方向的作品。
-   **约束 (Constraints)**：技术要求（框架、性能、无障碍性）。
-   **差异化 (Differentiation)**：什么能让它**令人难忘**？什么是用户一眼就能记住的记忆点？

**关键点**：选择一个清晰的概念方向并精准执行。无论是大胆的极繁主义还是精炼的极简主义，核心在于“意图感”，而非强弱。

实现的代码（HTML/CSS/JS, React, Vue 等）应具备：
-   生产级水平且功能完备
-   视觉震撼且具有记忆点
-   具有清晰审美视角的整体连贯性
-   每一个细节都经过精细打磨

## 前端审美准则 (Frontend Aesthetics Guidelines)

重点关注：

-   **字体排印 (Typography)**：选择美观、独特且有趣的字体。**严禁**使用 Arial 和 Inter 等平庸字体；应选择能提升前端美感的个性化字体。尝试将独特的标题字体与精致的正文字体配对。
-   **色彩与主题 (Color & Theme)**：坚持连贯的审美。使用 CSS 变量保持一致性。具有强烈视觉冲击的主色调搭配锐利的点缀色，效果远好于畏首畏尾、分布平庸的配色方案。
-   **动效 (Motion)**：使用动画增强微交互。HTML 优先使用纯 CSS 方案。React 推荐使用 Motion 库。关注高光时刻：一个精心编排、带有交错揭示（animation-delay）的页面加载过程，比零散的微交互更能带来惊喜。利用滚动触发和令人惊喜的悬停状态。
-   **空间构成 (Spatial Composition)**：打破常规的布局。不对称性、重叠、对角线流动、破格元素。慷慨留白或受控的密集感。
-   **背景与视觉细节 (Backgrounds & Visual Details)**：营造氛围和深度，而非默认使用纯色。添加符合整体审美的上下文效果和纹理。应用创意形式，如渐变网格、噪点纹理、几何图案、图层透明度、戏剧性的阴影、装饰性边框、自定义光标和颗粒感叠加。

**严禁**使用通用的 AI 生成式审美，例如：
-   滥用字体族（Inter, Roboto, Arial, 系统默认字体）。
-   陈词滥调的配色方案（尤其是白底紫光渐变）。
-   可预见的布局和组件模式。
-   缺乏具体情境特征的流水线式设计。

## 执行建议

进行创意解读，做出真正为特定情境设计的意外之选。拒绝千篇一律，在不同世代间尝试明暗主题切换、不同字体组合及审美流派。**永远不要**收敛到平庸的共同选择上（例如 Space Grotesk）。

**重要提示**：实现复杂度必须匹配审美愿景。极繁主义设计需要复杂的代码、丰富的动画和特效；而极简或精致的设计则需要克制、精准以及对间距、字体和细微环节的极致关注。优雅源于对愿景的完美执行。
