# 极致前端设计 (Frontend Aesthetics)

这是一个专门用于创作高保真、独具设计感的前端界面的技能插件。它旨在打破 AI 生成内容的平庸感，通过深度理解设计风格（从极简到极繁）来提供生产级的代码实现。

## 主要功能

- **高审美设计**：基于大胆的审美方向（如复古未来感、粗犷主义等）进行 UI 创作。
- **生产级代码**：交付可直接运行的、注重细节的 HTML/CSS/JS 或框架组件代码。
- **打破平庸**：严格遵循审美准则，避免使用常见的字体和色彩搭配。
- **动态交互**：集成高质量的动效和空间布局建议。

## 如何使用

当您需要构建以下内容时，可以调用此技能：
- 网页组件、仪表盘、落地页。
- 视觉海报、数字艺术品、移动端应用界面。
- 任何需要“设计感”而非单纯“功能实现”的前端 UI。

# Frontend Aesthetics Skill

This skill is dedicated to creating high-fidelity frontend interfaces with a strong sense of design. It aims to break the mediocrity of AI-generated content by providing production-grade code implementations based on deep understanding of design styles (from minimalist to maximalist).

## Key Features

- **High-Aesthetic Design**: UI creation based on bold aesthetic directions (e.g., Retro-futuristic, Brutalist, etc.).
- **Production-Grade Code**: Delivers runnable, detail-oriented HTML/CSS/JS or framework component code.
- **Beyond Generic**: Strictly follows aesthetic guidelines to avoid common font and color combinations.
- **Dynamic Interactions**: Integrates high-quality animations and spatial layout suggestions.

## How to Use

Invoke this skill when you need to build:
- Web components, dashboards, landing pages.
- Visual posters, digital artifacts, mobile app interfaces.
- Any frontend UI that requires "design quality" rather than just "functional implementation".
