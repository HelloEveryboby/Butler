# 截图即排障 (Screenshot Debugger)

这是 Butler 的核心排障技能。

## 功能
- **激光扫描：** 自动提取图片中的 Error/Exception。
- **智能诊断：** 识别端口冲突、数据库锁定、缺失依赖等。
- **一键修复：** 直接在界面弹出修复按钮。

## 使用
将报错截图拖入对话框即可触发。
