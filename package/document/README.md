# 文档处理工具包 (Document Package)

本目录包含文档编辑、格式转换和精准解析工具。

## 模块说明

- **TextEditor.py**: 简单的图形化文本编辑器。
- **markdown_converter.py**: 将各种文档格式转换为 Markdown 的工具。
- **marker_tool.py**: 深度文档结构化提取工具，支持 PDF、图片转 MD。
- **document_interpreter.py**: 文档智能分析器，支持对 PDF、Word 等进行摘要和 Q&A。
- **file_converter.py**: 通用文件格式转换工具。
- **excel_master_processor.py**: Excel 自动化处理与数据清洗工具。
- **translators.py**: 文本翻译工具。
