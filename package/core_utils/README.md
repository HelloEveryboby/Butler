# 核心工具包 (Core Utils Package)

本目录包含 Butler 系统的基础设施模块、调度器和辅助工具。

## 模块说明

- **log_manager.py**: 统一的日志管理系统。
- **dependency_manager.py**: 依赖安装与便携环境管理工具。
- **schedule_management.py**: 定时任务与日程管理系统。
- **embedding_utils.py**: 文本嵌入向量（Embedding）生成工具。
- **knowledge_base_manager.py**: 本地知识库（RAG）管理系统。
- **hybrid_orchestrator.py**: 混合系统（BHL）任务协调器。
- **autonomous_switch.py**: 自动交换机，负责进程治理与负载均衡。
- **thread.py**: 统一的线程池管理。
- **program_manager.py**: 动态程序加载与管理中心。
- **data_analyzer.py**: 数据分析辅助工具。
- **task_master/**: 任务大师 GUI 模块。
- **schedule.json**: 定时任务配置。
- **logging.txt**: 简单的日志记录输出。
