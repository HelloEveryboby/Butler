# 每日穿搭助手 (Daily Fashion Stylist)

## 简介
这是一个全能型的 Butler 技能，旨在通过综合分析实时天气、用户个人风格偏好以及今日的任务行程，为用户提供科学且具有美感的每日穿搭建议。

## 核心功能
- **全能体验**：不仅考虑天气，还结合 `HABITS.md` 中的个人画像和 `TASKS.md` 中的行程安排。
- **智能天气获取**：自动获取指定城市（或用户常驻城市）的实时天气数据。
- **结构化建议**：输出包含天气速览、风格定调、上下身详细搭配以及穿搭口诀。
- **风格匹配**：内置多种风格（通勤、休闲、温柔、街头等）的搭配公式。

## 安装与配置
该技能已默认集成并启用。

### 文件结构
```
daily_fashion_stylist/
├── manifest.json      # 技能元数据
├── __init__.py         # 核心逻辑入口
├── README.md           # 中文说明文档
├── SKILL.md            # AI 提示词参考
├── scripts/
│   └── get_weather.py  # 天气获取脚本
└── references/         # 规则与指南
    ├── style_guide.json
    └── weather_dressing_rules.json
```

## 使用示例
- “我今天穿什么好？”
- “在上海，我想穿得酷一点，帮我搭配一套。”
- “今天要见客户，给点穿搭建议。”

## 开发者说明
- **天气数据**：默认使用 `wttr.in` 服务。
- **LLM 驱动**：建议生成部分由 Butler 的 NLU 服务驱动，确保回复的人性化。
