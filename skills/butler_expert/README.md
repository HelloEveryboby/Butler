# Butler Expert Skill (butler_expert)

精通 HelloEveryboby/Butler 智能助手系统的架构、功能、部署与开发。

## 功能介绍

该技能旨在解答关于 Butler 项目的各方面技术问题，包括：
* **项目核心定位**：了解 Butler 的基本架构与设计理念。
* **核心功能模块**：AI 对话、多模式 UI、算法库、语音交互、本地知识库 (RAG) 等。
* **架构与工作流**：从输入到响应的完整命令处理流程。
* **故障排查**：针对 API、UI、插件、Excel 等常见问题的解决方案。

## 使用方法

### 1. 咨询项目信息
* **意图**：`ask`
* **示例**：
    * "Butler 是什么？"
    * "Butler 的核心功能有哪些？"
    * "Butler 是如何工作的？"

### 2. 详细功能说明
* **意图**：`explain`
* **示例**：
    * "详细说明一下 Butler 的现代 Web UI。"
    * "Butler 的 RAG 知识库使用了什么技术？"

### 3. 故障排查
* **意图**：`troubleshoot`
* **示例**：
    * "Web UI 无法访问怎么办？"
    * "API 调用报错如何排查？"
    * "列出常见的故障排查指南。"

## 开发扩展

Butler 采用模块化设计。如果您希望扩展功能：
1. **添加插件**：将 Python 脚本放置在 `plugin/` 目录下。
2. **添加包**：将独立工具放置在 `package/` 目录下。
3. **添加技能**：在 `skills/` 目录下创建新的文件夹并遵循开发规范。
