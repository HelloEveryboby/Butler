# Docker Manager (Docker 配置器)

## 定位
自动化运维助手。集成分配、下载、配置为一体的自动化系统。

## 功能
- **容器生命周期管理**: 自动执行容器的启动、停止、删除等操作。
- **动态配置**: 根据需求动态生成 Docker 配置脚本并自动部署。
- **镜像管理**: 自动化处理镜像的下载与分配。

## 权限说明
部署新容器或修改现有容器配置属于系统级高危操作，必须通过用户授权。

## 使用方法
1. 输入 `docker list` 查看所有容器状态。
2. 输入 `docker deploy image [镜像名]` 申请部署新服务。
