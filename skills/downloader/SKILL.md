# 🚀 资源下载器 (Resource Downloader)

高可用、多线程、跨端联动的全能下载套件。

## 核心特性
- **极速核心**：16-32 线程分块并发下载，充分跑满带宽，支持断点续传与 3 次智能重试。
- **协议全能**：支持 HTTP/HTTPS 协议，内置磁力/BT高保真仪表盘、迅雷/电驴协议解码、网盘链接模拟。
- **生态联动**：下载完成可一键无缝“转存”至 Storage Hub 聚合云盘或本地库。
- **智能嗅探与搜索**：内置网页媒体链接嗅探器，并集成多站聚合资源搜索。
- **实用工具箱**：边下边播（支持 HTML5 视频 Range 媒体流）、自动解压密码管理、系统 FFMPEG 格式转换、一键网络诊断与通知推送（Server酱/PushPlus）。
