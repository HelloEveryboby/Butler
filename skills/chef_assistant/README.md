# Chef Assistant (智慧厨师)

## 定位
个性化膳食顾问。根据食材储备、个人口味偏好，智能生成并推荐做菜方案。

## 功能
- **食材管理**: 查看当前库存（来源于 `data/chef_inventory.json`）。
- **智能菜谱推荐**: 结合剩余食材和口味权重算法，通过动态脚本生成最佳烹饪建议。
- **健康分析**: （扩展中）分析食材营养成分。

## 数据模型
使用 `data/chef_inventory.json` 存储食材（名称、数量、有效期）和用户偏好（口味权重、禁忌）。

## 使用方法
1. 输入 `chef list inventory` 查看冰箱剩余食材。
2. 输入 `chef recommend` 获取基于当前食材的智能菜谱建议。
